# log-hub
一个远程查看Log的工具, develop by Electron

### 目前支持
* UnityLog

![screenshot-1](https://github.com/aaBaO/loghub/blob/master/screenshots/screenshot-1.png?raw=true)

## How to use

### Test Unity Log
1. Add the test script in `unity-test` folder to `YourUnityProject`
2. Run `npm start`
